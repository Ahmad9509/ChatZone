# Simple Page with Top and Bottom Bars

A simple React component built with TypeScript that displays a page with a top navigation bar and bottom footer bar.

## Files

- `SimplePage.tsx` - Main React component
- `SimplePage.css` - Styling for the component

## Features

- ✅ Top navigation bar with links
- ✅ Bottom footer bar with copyright and links
- ✅ Responsive design (mobile-friendly)
- ✅ Modern, clean styling
- ✅ Hover effects on links

## How to Use

1. Install dependencies:
```bash
npm install react react-dom
npm install --save-dev @types/react @types/react-dom typescript
```

2. Import and use the component:
```tsx
import SimplePage from './SimplePage';

function App() {
  return <SimplePage />;
}
```
